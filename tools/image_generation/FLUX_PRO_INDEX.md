## ‚úÖ Quality Gates

Before marking any asset as "complete":

### Design Review
- [ ] Colors match SpiritAtlas palette (eyedropper check)
- [ ] Style consistent with other assets in category
- [ ] Composition balanced and professional
- [ ] Sacred symbols culturally accurate and respectful

### Technical Review
- [ ] Correct dimensions (verify in image properties)
- [ ] Appropriate file size (<500KB backgrounds, <200KB icons)
- [ ] Proper format (PNG with alpha or WebP)
- [ ] No generation artifacts or errors

### Integration Review
- [ ] Works on dark background
- [ ] Text remains readable (if applicable)
- [ ] Scales well at target display sizes
- [ ] Android density variants created

### Approval
- [ ] Designer sign-off
- [ ] Developer integration test
- [ ] Accessibility check (contrast, clarity)
- [ ] Final stakeholder approval
