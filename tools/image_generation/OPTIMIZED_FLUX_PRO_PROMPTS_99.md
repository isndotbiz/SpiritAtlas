## USAGE NOTES

### FLUX 1.1 Pro Optimization Tips:
1. **Aspect Ratios**: Use exact dimensions for best results
2. **Color Codes**: Include hex codes for color accuracy
3. **Style Keywords**: Photography terms (8K, ultra-detailed, professional) improve quality
4. **Lighting**: Specify lighting (studio, rim, glow) for depth
5. **Composition**: State centered, symmetrical, balanced for UI elements

### Batch Generation Workflow:
1. Generate **Category 1 (Branding)** first - establishes visual identity
2. Generate **Category 3 (Avatars)** and **Category 9 (UI Elements)** - highest usage
3. Generate **Category 4 (Zodiac)** and **Category 5 (Chakras)** - core content
4. Generate remaining categories based on development priority

### Post-Processing:
- Remove backgrounds for icons/avatars (use PNG transparency)
- Optimize file sizes for mobile (WebP format for backgrounds)
- Create @2x and @3x versions for Android density buckets
- Test legibility at target sizes before finalizing
