**Coil Caching Strategy**:
```kotlin
ImageLoader.Builder(context)
    .memoryCache {
        MemoryCache.Builder(context)
            .maxSizePercent(0.25) // 25% app memory = ~40MB cache
            .strongReferencesEnabled(true)
            .build()
    }
    .diskCache {
        DiskCache.Builder()
            .directory(context.cacheDir.resolve("image_cache"))
            .maxSizePercent(0.02) // 2% disk = ~200MB on 10GB device
            .build()
    }
    .build()
```

**Performance Targets**:
- ‚úÖ First image load: <100ms (with caching)
- ‚úÖ Cached load: <10ms

2. Multi-density (only loads appropriate density)
