### Active Documents
- Updated regularly
- Located in root or primary directories
- Version tracked in git
- Referenced in this index

### Archived Documents
- Historical reference only
- Located in docs/archive/
- Not actively maintained
- Dated with archive date

### Naming Conventions
- ALL_CAPS.md for major documents
- lowercase-with-hyphens.md for supporting docs
- Date format: YYYY-MM-DD in filenames when needed
