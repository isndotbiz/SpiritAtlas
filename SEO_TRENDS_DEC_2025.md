   - Store listing visit ‚Üí install

4. **Ratings & Reviews (10%)**

- ‚ùå Don't: Repeat keywords unnaturally in description

**Fake Reviews:**
- ‚ùå Don't: Buy reviews or incentivize 5-star ratings
- ‚úÖ Do: Earn organic reviews through exceptional UX

- ‚ùå Don't: Show features that don't exist

- ‚ùå Don't: Let app stagnate for months
