## General Style Guidelines

**Core Aesthetic:**
- Mystical, ethereal, cosmic, spiritual
- High-quality digital art with depth
- Glowing elements, subtle light effects
- Sacred geometry influences
- Colors: Deep purples (#7B2CBF), cosmic blues (#4361EE), golden accents (#FFB627), whites
- Professional, polished, modern spiritual design
- NOT cartoony, NOT overly busy, NOT clipart style

**Technical Requirements:**
- Resolution: Specified per category below
- Format: PNG with transparency OR WebP for backgrounds
- Color space: sRGB
- Bit depth: 24-bit color + 8-bit alpha
- No watermarks, no text (except where specified)

## Image Optimization Post-Generation

### Conversion Commands
```bash
# Convert PNG to WebP (lossless)
cwebp -lossless input.png -o output.webp

# Convert PNG to WebP (quality 90)
cwebp -q 90 input.png -o output.webp

# Batch convert
for file in *.png; do cwebp -q 90 "$file" -o "${file%.png}.webp"; done
```

### Size Guidelines
- **Icons (512x512):** Target < 100KB
- **Backgrounds (1920x1080):** Target < 500KB
- **Buttons (320x100):** Target < 50KB
- **Use:** `-resize` flag if needed: `cwebp -resize 512 512 input.png -o output.webp`
