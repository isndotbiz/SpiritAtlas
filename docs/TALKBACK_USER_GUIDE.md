# TalkBack User Guide for SpiritAtlas

> Navigate your spiritual journey with confidence using TalkBack

## What is TalkBack?

TalkBack is Android's built-in screen reader that speaks text and describes actions on your screen. With TalkBack enabled, you can navigate SpiritAtlas entirely by touch and sound.

## Enabling TalkBack

### Method 1: Quick Settings
1. Press and hold both **Volume Up** and **Volume Down** keys for 3 seconds
2. TalkBack will announce "TalkBack on" or "TalkBack off"

### Method 2: Settings Menu
1. Open **Settings** on your Android device
2. Go to **Accessibility**
3. Select **TalkBack**
4. Toggle the switch to **On**
5. Tap **Allow** when prompted

## Basic TalkBack Gestures

### Navigation
- **Swipe Right**: Move to next item
- **Swipe Left**: Move to previous item
- **Swipe Down then Right**: Read from top of screen
- **Swipe Up then Right**: Read from current position

### Actions
- **Double-Tap**: Activate the selected item
- **Double-Tap and Hold**: Long press action
- **Two-Finger Tap**: Pause/resume TalkBack speech
- **Three-Finger Swipe Up/Down**: Scroll page

### Text Editing
- **Swipe Right then Left**: Navigate by character
- **Swipe Left then Right**: Navigate by word
- **Two-Finger Swipe Up/Down**: Change reading granularity

## Customizing TalkBack

### Speech Settings
1. Open **TalkBack Settings**
2. Select **Text-to-speech settings**
3. Adjust:
   - **Speech rate**: Slow, Normal, Fast
   - **Pitch**: Lower or higher voice
   - **Language**: Choose your language

### Verbosity Settings
1. In **TalkBack Settings**
2. Select **Verbosity**
3. Choose level:
   - **High**: Maximum detail
   - **Low**: Essential information only

### Customize Gestures
1. Open **TalkBack Settings**
2. Select **Customize gestures**
3. Assign frequently used actions

## Troubleshooting

### TalkBack Not Speaking

**Solution 1**: Check volume
- Press **Volume Up** button
- Ensure media volume is not muted

**Solution 2**: Restart TalkBack
- Hold both volume keys for 3 seconds (toggle off)
- Hold both volume keys again for 3 seconds (toggle on)

**Solution 3**: Check language settings
- Settings > Language & input > Text-to-speech
- Ensure correct language is selected

### Can't Activate Items

**Problem**: Single tap doesn't work
**Solution**: Use **double-tap** to activate

**Problem**: Wrong item activates
**Solution**:
1. Ensure TalkBack announces the correct item
2. Wait for announcement to complete
3. Then double-tap

### Reading Order Confusing

**Solution**: Use heading navigation
1. **Swipe Up then Down** to jump between sections
2. Use **Swipe Right** for detailed exploration
3. **Swipe Down then Right** to start from top

### Form Entry Difficult

**Solution 1**: Use explore by touch
- Drag finger to find field
- Lift finger
- Double-tap to activate

**Solution 2**: Navigate systematically
- **Swipe Right** through all fields
- **Double-tap** on desired field
- Use on-screen keyboard
