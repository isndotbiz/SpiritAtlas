## Emergency Contacts & Resources

**If Something Goes Wrong:**

**Critical Bug on Launch Day:**
1. Document the issue with screenshots
2. Check crash reports in Play Console
3. If affecting >5% of users, consider temporary delisting
4. Push hotfix update ASAP
5. Communicate with users via social media

**Negative Reviews:**
1. Respond within 24 hours
2. Thank user for feedback
3. Offer solution or workaround
4. Invite them to contact support directly
5. Never argue or get defensive

**App Store Rejection:**
1. Read rejection reason carefully
2. Fix the specific issue mentioned
3. Resubmit with detailed explanation
4. Common issues: Privacy policy, inappropriate content rating, misleading screenshots

**Social Media Crisis:**
1. Don't delete negative comments (unless spam/abusive)
2. Respond calmly and professionally
3. Take conversation to DM if needed
4. Document everything
5. Learn from feedback
